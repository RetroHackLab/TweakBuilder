# TweakBuilder v1.3.0 🚀

A production-hardened iOS tweak development environment orchestrator and structural injector designed by **RetroHackLab (2026)**. This tool automates the generation of control configurations, directory layouts, PreferenceBundles, Substrate filters, and deployment Makefiles for legacy iOS development.

## 🛠️ Workspace Architecture

The framework is designed to link seamlessly with custom toolchains and flexible source file configurations:

```text
📂 Your_Project_Workspace/
├── 📄 build.sh          # Core automation script (v1.3.0)
├── 📄 Clangy.sh         # Your custom Clang compilation layer
├── 📄 DEBIAN.sh         # Your custom Debian dpkg-deb packaging tool
├── 📄 SDK_INSTALL.sh    # Pre-requisite script to download and install Theos SDKs
├── 📂 sdks/             # Local repository containing iOS SDKs (e.g., iPhoneOS9.3.sdk)
├── 📄 Makefile          # Dynamically generated by build.sh
└── 📂 Package/          # Staged root filesystem directory for the final .deb package
